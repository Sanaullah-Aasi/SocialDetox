# SocialDetox 📵

A productivity Flutter app that helps you reduce screen time by blocking internet access for selected apps.

## Features

- 🚫 Block internet access for distracting apps
- ⏱️ Set custom blocking schedules
- 📊 Track your app usage
- 🎯 Stay focused and productive

## Tech Stack

- **Framework:** Flutter
- **State Management:** Provider
- **Local Storage:** SharedPreferences
- **UI:** Material Design with Google Fonts

## Getting Started

### Prerequisites

- Flutter SDK ^3.7.2
- Android Studio / VS Code
- Android device or emulator

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/Sanaullah-Aasi/SocialDetox.git
   cd SocialDetox
   ```

2. Install dependencies:
   ```bash
   flutter pub get
   ```

3. Run the app:
   ```bash
   flutter run
   ```
